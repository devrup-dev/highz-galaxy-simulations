# Anatomy of an Enzo Parameter File

An Enzo parameter file controls:
1. Cosmology
2. Grid structure
3. Physics modules
4. Output & runtime control

## 1. Cosmology Block
Controls expansion of the universe, redshift range, and background cosmology.

## 2. Grid and Resolution
Controls how finely space is resolved.

## 3. Physics Modules
Hydrodynamics, cooling, star formation, feedback.

## 4. Outputs
How often data is written.
